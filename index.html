libsegwayrmp

This is a C++ Interface for Segway's Robotic Mobile Platforms (RMP's).

Download this project as a .zip file Download this project as a tar.gz file

This is a cross-platform, open source interface to Segway's Robotic Mobile Platforms. It provides access to the interface of the RMP50, 100, 200, and 400 series via either serial rs-232 (VCP) or ftdi usb mode (D2XX).

Dependencies

  • CMake, for the build system: http://www.cmake.org/
  • Boost, for threading: http://www.boost.org/
  • FTDI D2XX (optional), for ftd2xx based usb support: http://www.ftdichip.com/Drivers/D2XX.htm
    • Manual installation only required on OS X
  • Serial (optional), for serial based communication: https://github.com/wjwwood/serial

Installation

Get the source:

git clone git://github.com/segwayrmp/libsegwayrmp.git
cd libsegwayrmp

Compile the code:

make

Or run cmake youself:

mkdir build && cd build
cmake ..
make

Install the code (UNIX):

make
sudo make install

Uninstall the code (UNIX):

make
sudo make uninstall

Run the test (Requires GTest):

make test

Build the documentation:

make doc
